Electrical

PCBs
The PCB created for the main microcontroller, the motor controller, inductive sensor, encoders, and a few I/O expanders can be found in the drive below. We chose to create PCBs for most of our main circuits because debugging our circuits and using connectors would be easier than hand soldering all the components to a perfboard.
